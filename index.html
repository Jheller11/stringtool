String Tool

About this page

This is a small project to accomplish a comparison between two strings. In the comparison, I'm looking to split the strings on any space or comma, then identify which terms are unique to each string and which terms are shared between the two. This helps me troubleshoot issues more quickly by ensuring that required terms are present and potentially identifying areas for further investigation.
